# LatihanVCS

Langkah 1: Instalasi Git
Pertama, pastikan Anda menginstal Git pada komputer Anda. Anda dapat mengunduh Git dari situs resmi (https://git-scm.com/), lalu ikuti langkah-langkah instalasi yang sesuai dengan sistem operasi Anda.

Langkah 2: Konfigurasi Git
Setelah Git diinstal, Anda perlu mengkonfigurasi nama pengguna dan alamat email Anda. Ini diperlukan untuk mengidentifikasi kontribusi Anda dalam repositori. Buka terminal (pada Windows, gunakan Git Bash) dan jalankan perintah berikut:

Anda dapat memeriksa status perubahan yang belum di-commit dan riwayat komit dengan perintah:

Langkah 7: Menarik dan Menggabungkan (Pull and Merge)
Jika Anda bekerja dalam proyek yang berkolaborasi, Anda perlu menarik (pull) perubahan dari repositori jarak jauh (remote repository) dan menggabungkannya dengan cabang Anda. Misalnya:
<img width="475" alt="Cuplikan layar 2023-10-16 174949" src="https://github.com/MuhammadPrayoga/LatihanVCS/assets/147376352/950a7483-a8c6-436d-806c-a7a0878e37cf">

Langkah 8: Mengunggah (Push) Perubahan
Jika Anda telah menyelesaikan pekerjaan Anda di cabang fitur-baru dan ingin mengunggah perubahan Anda ke repositori jarak jauh, gunakan perintah berikut:
<img width="478" alt="Cuplikan layar 2023-10-16 175121" src="https://github.com/MuhammadPrayoga/LatihanVCS/assets/147376352/2e6bd450-38cd-4b5a-844e-75000ffcf987">

Langkah 9: Perbandingan (Diff) dan Penyesuaian (Resolve)
Jika ada konflik saat menggabungkan cabang, Anda perlu menyelesaikan konflik tersebut dengan memodifikasi file yang berselisih (biasanya diidentifikasi oleh Git). Setelah menyelesaikan konflik, komit perubahan tersebut.

Langkah 10: Menggabungkan Cabang
Setelah Anda selesai dengan fitur atau perbaikan, Anda dapat menggabungkan cabang Anda ke cabang utama (misalnya, "master"):
